# Telegram Бот для учёта счетов, заявок, ТМЦ, МФУ, почты и договоров

## Описание

Бот объединяет все разделы:
- Счета к заявкам и счета за связь
- Заявки и быстрые напоминания
- ТМЦ 50/50
- МФУ
- Генерация договоров
- Работа с почтой (Mail.ru бизнес)
- Проверка оплат
- Отчёты на месяц
- Глобальный поиск по всем разделам

Бот работает как в личных сообщениях, так и в группе.
